# Sugar Rush 1000 — Frontend Application

This directory contains the Phaser 3 / TypeScript frontend for the Sugar Rush 1000 slot game clone.

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
